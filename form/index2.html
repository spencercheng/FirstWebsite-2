CSS Lists

Today, we are going to be learning about Lists in CSS

  • list-style type
  • list-style image
  • list-style position

List-style type

In CSS, you can change or alter the way the list is presented (ex: the bullet points in the unordered list)

Here are examples of how the code would be written

  • list-style-type: upper-roman;
  • list-style-type: lower-greek;
  • list-style-type: lower-alpha;
  • list-style-type: simp-chinese-formal;

If you want to check out more list-style-types, you can find more examples on mdn.

List-style image

This style can change the bullet point(marker) to an image.

  • list-style-image:url(example.png);

List-style position

There are two main types of List-Style Positions

  • list-style position:outside; where the bullet point(marker) will be on the outside of the highlighted list item. It means that it will be outside the block box.
  • list-style position:inside; where the bullet point(marker) will be on the inside of the highlighted list item, it will be apart of the text.It
